## 🎯 О проекте
Интерактивный анимированный плакат, состоящий из трёх игровых модулей:

«Таинственные узоры: угадай животное» – вращающийся калейдоскоп, где нужно угадать животное по узору.

«Волшебная память: раскрась узор» – раскрашивание SVG-узорчика с запоминанием исходных цветов и проверкой.

«Загадочные животные: послушай голос» – колесо фортуны, после вращения которого можно услышать голос выпавшего животного.

Проект разработан с акцентом на интерактивность, адаптивность и плавную анимацию.

## 🤖 Роль нейросети DeepSeek в разработке

Некоторые сложные логические блоки были написаны с помощью генерации кода и консультаций с нейросетью DeepSeek. Это позволило:

Быстро реализовать алгоритмы анимации и проверки состояний.

Сократить время на отладку и тестирование интерактивных сценариев.

## 🧩 Самые сложные функции (реализованы с участием DeepSeek)

### 1. Раскраска SVG с эталонными цветами
Функция getAllFillElements рекурсивно собирает все элементы с атрибутом fill.

saveOriginalColors сохраняет исходные цвета, игнорируя fill="none".

Проверка checkColoring сравнивает текущий цвет с эталоном и запускает мигание красным для ошибочных элементов.

Таймер whitenPattern обесцвечивает узор через 5 секунд после начала игры, активируя возможность раскрашивания.

Курсор загружается из brush.svg, при выборе цвета в палитре обновляется его заливка (updateCursorColor). Курсор показывается только в области .pattern-area.

### 2. Колесо с точной остановкой в центре сектора
Рассчитывается угол центра каждого сектора с учётом начального положения указателя (внизу).

Генерация случайного числа полных оборотов + доворот до центра выбранного сектора (getSectorCenterAngle, spinWheel).

Используется transitionend для фиксации результата после анимации.
